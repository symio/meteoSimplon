## Configuration

- créez une clé d'api météo-concept
- copiez le fichier public_html/config/configuration.sample.json vers public_html/config/configuration.json
- adaptez la configuration en ajoutant
> 1. Votre clé d'api Météo-concept
> 2. à minima le nom de la ville
> 3. sont code insee que vous trouverez en recherchant "code insee" suivi du nom de la ville dans Google
